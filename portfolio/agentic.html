Velyana – AI-Powered SaaS Platform for Business Automation

Velyana is a cloud-based AI SaaS platform that unifies automation, multimodal document understanding, and conversational intelligence into a single ecosystem. Users interact with a prompt-driven multi-agent system capable of processing text, PDFs, images, and voice inputs securely. The platform supports SaaS distribution, allowing administrators to create accounts, provision services, and manage user access across multiple clients.

Key Features:

  • Multi-Agent Architecture:
    • Centralized prompt routing engine to dispatch instructions to the correct agent.
    • Scalable multi-tenant system for isolated service instances per client.
  • Quote Agent:
    • Processes any type of project data — PDFs, images, and text — to generate detailed quotations and cost breakdowns.
    • Uses OCR, vision-language modeling, and contextual reasoning for accurate outputs.
    • Includes security and prompt protection to sanitize and validate user input.
    • Powered by fine-tuned Hugging Face models for robust performance.
  • BOQ (Bill of Quantities) Agent:
    • Interprets project specifications in any format to produce structured BOQs automatically.
    • Moderation ensures outputs meet safety and compliance requirements.
    • Uses custom fine-tuned Hugging Face models for domain-specific understanding.
  • Flight Booking Agent:
    • Handles end-to-end flight booking workflows via natural-language prompts.
    • Integrates with live flight APIs and supports secure, multi-tenant usage.
  • Voice Agents:
    • AI-powered assistants capable of making or receiving calls using speech recognition and synthesis.
    • Enforces prompt moderation and secure handling of sensitive information.
    • Can be provisioned individually per SaaS account.
  • SaaS Administration:
    • Admin can create accounts, distribute AI services, and manage client access.
    • Supports logging, monitoring, and safe multi-tenant operation.

Results / Impact:

  • Built a secure, moderated AI SaaS ecosystem with multi-agent orchestration.
  • Enabled automated processing of text, PDFs, images, and voice in real-time workflows.
  • Empowered admins to distribute services safely to multiple clients while retaining control.

Technologies Used:

Python, FastAPI, PyTorch, LangChain, Hugging Face (fine-tuned models), OCR, Vision-Language Models, Whisper, RESTful APIs, Multi-Agent Orchestration, Security & Prompt Moderation Modules, SaaS Management Tools.

Unique Contributions:

  • Designed and implemented multi-agent orchestration for SaaS deployment.
  • Built multimodal pipelines for secure text, PDF, image, and voice processing.
  • Integrated Hugging Face fine-tuned models for domain-specific task performance.
  • Developed SaaS features for account creation, service distribution, and secure multi-tenant management.
