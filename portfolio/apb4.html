GPIO and UART Modules for APB Interface

This project involved the design, implementation, and optimization of GPIO and UART modules for the AMBA APB protocol. The project demonstrates expertise in HW systems and digital design.

Key Features:

  • GPIO Module:
    • Fully parameterized, supporting bidirectional I/O.
    • Configurable Push-Pull and Open-Drain modes.
    • Error detection via PSLEVRR signal.
    • Optimized performance, increasing clock frequency from 145 MHz to 661 MHz.
  • UART Module:
    • Supports parallel-to-serial and serial-to-parallel data conversion.
    • Includes baud rate generation and parity support.
    • Finite State Machine for reliable communication.
  • APB Interface:
    • Efficient read/write transfers with and without wait states.
    • Low-complexity signal handling for easy peripheral integration.

Testing and Results:

  • Comprehensive testing of GPIO for different modes and synchronization.
  • Validated UART communication with parity and multi-state FSM.
  • Successfully integrated APB read/write operations, ensuring reliability.

Technologies Used:

Verilog HDL, Xilinx Vivado, AMBA APB Protocol.

Unique Contributions:

  • Optimized GPIO module for enhanced performance.
  • Developed testbenches to validate all functionalities and error scenarios.
  • Implemented synchronization mechanisms to ensure stable I/O operations.
