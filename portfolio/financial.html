Personal Finance Manager

This project is a full-stack personal finance management web application developed using Spring Boot for the backend and Next.js for the frontend. The system helps users track income, expenses, and savings while providing insightful reports and visualizations. The backend ensures secure data handling with RESTful APIs, while the frontend delivers a modern, responsive user experience. Key features include:

  • Expense & Income Tracking: Users can log daily expenses and income, categorized for better financial visibility.
  • Budget Management: Allows setting monthly or custom budgets with real-time progress tracking against spending limits.
  • Interactive Dashboards: Next.js frontend displays interactive charts and graphs for cash flow, savings trends, and expense breakdowns.
  • Spring Boot Backend: Provides secure and scalable REST APIs for authentication, transactions, and analytics.
  • Data Storage: Uses PostgreSQL/MySQL for persistent storage of user accounts, financial records, and reports.
  • Authentication & Security: Implements JWT-based authentication to ensure data privacy and user isolation.

The application empowers users to gain control of their personal finances by turning raw transaction data into actionable insights. With budget alerts, categorized expense tracking, and visually appealing dashboards, it simplifies money management and helps users make informed financial decisions.
