Regression with Insurance Dataset

This project focuses on predicting insurance charges using multiple machine learning approaches, including Self-Supervised Neural Networks, Random Forest Regression, and CatBoost. The goal is to identify the most effective model for estimating medical insurance costs.

Key Features:

  • Data Preprocessing:
    • Handled missing values and outliers for clean and consistent data.
    • Performed feature engineering, such as encoding categorical variables and scaling numerical features.
  • Model Development:
    • Explored Self-Supervised Neural Networks, Random Forest, and CatBoost models.
    • Optimized each model using techniques like GridSearchCV for hyperparameter tuning.
  • CatBoost Implementation:
    • CatBoost achieved the highest accuracy and lowest error rates.
    • Efficiently handled categorical variables without requiring extensive preprocessing.
  • Model Evaluation:
    • Evaluated models using metrics such as Mean Absolute Error (MAE), Mean Squared Error (MSE), and R-squared.
    • Visualized residuals and prediction distributions for insights into model performance.

Results:

  • CatBoost outperformed other models, delivering the most accurate predictions.
  • Provided actionable insights into factors driving insurance costs, such as age, BMI, and smoking status.

Technologies Used:

Python, CatBoost, Random Forest, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn.

Unique Contributions:

  • Developed and compared multiple machine learning approaches to identify the optimal model.
  • Designed an efficient preprocessing pipeline for handling real-world data challenges.
  • Presented detailed evaluations and visualizations to communicate results effectively.
