Microwave Controller

This project involves developing a TIVA-C microwave oven controller with custom drivers built from scratch. Key features of this platform include:

  • Custom Drivers: Implementation of custom drivers for controlling the microwave.
  • TIVA-C Microcontroller: Utilizes the TIVA-C microcontroller for managing the microwave functions.
  • Efficient Control: Provides efficient and precise control over microwave operations.

The project focuses on building the microwave controller from the ground up, ensuring that all drivers are customized to meet specific requirements. This allows for greater control and optimization of the microwave's performance.
