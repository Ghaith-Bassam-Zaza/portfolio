NLP with Disaster Tweets Using RoBERTa

This project leverages the RoBERTa transformer model to classify disaster-related tweets. By fine-tuning the pre-trained RoBERTa model on a dataset of tweets, the project aims to identify whether a given tweet pertains to a disaster or not.

Key Features:

  • Data Preprocessing:
    • Cleaned and tokenized tweet text to prepare it for model input.
    • Utilized the Hugging Face tokenizer for compatibility with RoBERTa.
  • Model Fine-Tuning:
    • Fine-tuned the pre-trained RoBERTa base model on the disaster tweet dataset.
    • Implemented techniques to handle class imbalance, such as weighted loss functions.
  • Model Evaluation:
    • Assessed performance using metrics like F1 score, accuracy, and precision.
    • Visualized results through confusion matrices and classification reports.

Results:

  • Achieved high classification accuracy and F1 score, demonstrating the effectiveness of RoBERTa for this task.
  • Provided insights into common patterns in disaster-related tweets.

Technologies Used:

Python, Hugging Face Transformers, PyTorch, Pandas, NumPy, Matplotlib.

Unique Contributions:

  • Fine-tuned a state-of-the-art transformer model for text classification.
  • Developed a robust pipeline for preprocessing and evaluating text data.
  • Created informative visualizations to communicate model performance.
