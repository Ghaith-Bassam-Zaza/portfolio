Titanic Survival Prediction Using Machine Learning

This project involves predicting the survival of passengers aboard the Titanic using machine learning models. By analyzing the Titanic dataset, the project explores various factors affecting survival and builds predictive models to classify outcomes.

Key Features:

  • Data Preprocessing:
    • Handled missing data and outliers to improve dataset quality.
    • Engineered new features, such as family size and title extraction, to enhance model performance.
  • Model Development:
    • Implemented Logistic Regression, Random Forests, and Gradient Boosting classifiers.
    • Utilized GridSearchCV for hyperparameter tuning.
  • Model Evaluation:
    • Assessed model performance using metrics like accuracy, precision, and recall.
    • Visualized results with ROC curves and confusion matrices.

Results:

  • Achieved high accuracy with Gradient Boosting and Random Forest models.
  • Identified key factors influencing survival, such as passenger class and age.

Technologies Used:

Python, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn.

Unique Contributions:

  • Developed an end-to-end pipeline for preprocessing, modeling, and evaluation.
  • Performed feature engineering to improve model accuracy.
  • Created detailed visualizations to communicate findings effectively.
