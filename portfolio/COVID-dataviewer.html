COVID-19 Data Analyser - Matlab GUI Application

This application visualizes COVID-19 pandemic data from Johns Hopkins University, tailored for analysing and displaying relevant statistics. Key features include:

  • Country and State Selection: Two list boxes allow users to select a country and its corresponding states or territories (if applicable). "Global" is the default option.
  • Interactive Plot: Displays a bar chart of the cumulative number of cases, defaulting to global data. Titles and plot content dynamically update based on selected country/state.
  • Data Types: Users can switch between cumulative confirmed cases, daily confirmed cases, and deaths. Both cumulative and daily data can be plotted.
  • Dual Y-axes: The application uses dual y-axis scales for plotting both cases and deaths in a single figure, facilitating a comprehensive comparison.
  • Data Averaging: An adjustable slider smooths out daily data by averaging over different numbers of days (1, 7, 15, etc.), enhancing clarity.
  • Object-Oriented Design: Utilizes object-oriented programming to encapsulate country and state data, ensuring efficient and clear data management.
