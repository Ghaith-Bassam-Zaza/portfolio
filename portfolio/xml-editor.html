XML Editor & Social Network Analysis Project

This project is a versatile XML editor designed for string processing tasks on XML, such as checking, formatting, compressing, and converting XML to JSON. Additionally, it offers functionalities to analyze a social network and represent it as a graph.

Developed with a user-friendly interface, the application allows users to easily manage XML files and perform complex operations without needing extensive technical knowledge. The social network analysis feature provides graphical visualizations, making it easier to understand and interpret complex relationships within a network.

The backend is built to efficiently handle data processing tasks, ensuring fast performance and accurate results. The tool supports various XML operations, from basic checks to advanced manipulations.

Key features of the project include XML validation, formatting, compression, conversion to JSON, and social network analysis. Users can visualize social networks as graphs, providing valuable insights into the structure and dynamics of the network.
